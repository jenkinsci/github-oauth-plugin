The Embeddable Build Status Plugin allows Jenkins to expose the current status of your build as an image in a fixed URL. You can put this URL into other sites (such as GitHub README) so that people can see the current state of the build.

By enabling this permission you grant anonymous external READ access to the /job/*/badge/icon URLs so that the icon is accessible.
e.g. http://localhost:8080/jenkins/job/jobName/badge/icon
