Normally this value stays as-is. If you are using GitHub Enterprise then you should enter the URI to the API root of your GitHub installation (e.g. https://github.company.com/api/v3).

Notes:

  1. The https:// / http:// part needs to be specified.
  2. There should not be any trailing slash (/)
  3. For GitHub Enterprise, the API URI is usually the same as the Web URI, with /api/v3 appended.
